# 📊 Análisis de Embudo y Retención – MercadoLibre

## 📌 Contexto del proyecto
Este proyecto analiza el comportamiento de usuarios dentro de un embudo de conversión y su retención a lo largo del tiempo en un entorno tipo e-commerce (MercadoLibre).

El objetivo es entender cómo los usuarios avanzan desde la interacción inicial hasta la conversión final, así como evaluar la capacidad de retención de la plataforma.

---

## 🎯 Objetivos

- Analizar el embudo de conversión
- Identificar puntos de abandono
- Evaluar la retención de usuarios
- Generar insights accionables para mejorar la conversión

---

## ⚙️ Proceso de análisis

1. **Extracción de datos**
   - Datos de interacción de usuarios

2. **Limpieza de datos**
   - Validación de inconsistencias
   - Estandarización

3. **Transformación**
   - Construcción del embudo
   - Cálculo de tasas de conversión

4. **Análisis**
   - Identificación de cuellos de botella
   - Análisis de retención

5. **Visualización**
   - Representación gráfica del funnel
   - Curvas de retención

---

## 🔍 Análisis del embudo

El embudo permite visualizar el recorrido del usuario:

1. Visitas
2. Interacción
3. Agregado al carrito
4. Compra

### 📉 Hallazgos
- Existe una caída significativa en etapas intermedias
- El mayor abandono ocurre antes de la conversión final

---

## 🔄 Análisis de retención

Se evaluó la permanencia de usuarios en el tiempo:

- Retención inicial alta
- Disminución progresiva en periodos posteriores

### 📈 Insights
- Necesidad de estrategias de fidelización
- Importancia de mejorar la experiencia inicial
